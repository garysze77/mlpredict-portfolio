# MLPredict Logo 與 Favicon 設計說明

## 1. 品牌資訊

- **名稱**：MLPredict  
- **Tagline**：Machine Learning Systems Development  
- **定位**：專做 ML 預測系統（體育、股票、物流等），偏技術向、專業、穩陣。

---

## 2. 三個 Logo 概念（文字描述）

### 概念 A：折線圖 + 圓點節點（圖標在左）

**形狀與比例**
- 左邊為一獨立圖標：一條由左下往右上延伸的折線（3 段線、4 個節點），線條粗度約 2px，節點為小圓點（直徑約為線粗的 2.5 倍）。
- 折線整體在一個隱性方框內，寬高比約 1:1，可視為 24×24 單位；折線起點在左下約 (2,20)，經 (8,14) → (14,8) → (20,2)，營造「上升趨勢／預測」感覺。
- 右邊為文字 "MLPredict"，字重 medium 或 semibold，字高與圖標高度一致（例如圖標 24px 高，字高 24px），字與圖標間距約 8px。
- 水平 Logo 總寬度約：圖標 24px + 間距 8px + 文字寬度。

**顏色（十六進位）**
- 主色（折線 + 節點 + 文字）：`#0ea5e9`（primary blue，與網站一致）
- 淺色背景 / 反白用：折線與節點改為 `#ffffff`，背景為上述藍或深色

**用在 Header 的樣子**
- 水平排列：\[圖標]  MLPredict
- Header 背景白時用主色 `#0ea5e9`；若 Header 有深色區塊，可改為白字 + 白圖標。

**縮成 32×32 Favicon 要保留的元素**
- 只保留左邊「折線 + 4 個節點」圖標，可置於圓形或圓角方背景內以利辨識。
- 捨棄文字 "MLPredict"，否則 32×32 會難以辨讀。

---

### 概念 B：字母 M + 簡單折線（M 與圖合一）

**形狀與比例**
- 將 "M" 的右半或整體與一條上升折線結合：M 的兩條斜邊可視為折線的前兩段，第三段從 M 的谷底向右上延伸，形成「M + 趨勢線」一體圖形。
- M 的頂部兩角與折線的末端節點可共用或微調為小圓點，線條統一 2px 粗。
- 圖標寬高比約 1:1（如 24×24），右側同樣接 "MLPredict" 文字，字高與圖標同，間距約 8px。

**顏色（十六進位）**
- 主色：`#0369a1`（較深藍，顯專業）
- 輔助／hover：`#0ea5e9`
- 反白：圖標與文字 `#ffffff`

**用在 Header 的樣子**
- 水平排列：\[M+折線圖標]  MLPredict
- 深色 Header 用白線 + 白字；淺色 Header 用主色 `#0369a1`。

**縮成 32×32 Favicon 要保留的元素**
- 保留「M + 第三段上升線」的整體輪廓，可略簡化 M 的細節（例如谷底改為一折），確保在小尺寸下仍能看出 M 與上升趨勢。
- 捨棄文字。

---

### 概念 C：圓形容器 + 極簡折線（僅 3 點 2 段）

**形狀與比例**
- 圖標為一正圓，圓內一條極簡折線：僅 3 個點、2 段線（左下 → 中上 → 右上），線粗約 1.5px，節點為小圓點。
- 圓直徑與文字 "MLPredict" 字高同（如 24px），圓與文字間距約 8px。
- 整體偏「數據／預測」感，圓形象徵完整、穩定。

**顏色（十六進位）**
- 圓形描邊或背景：`#0ea5e9`（或透明描邊）
- 折線 + 節點：`#ffffff`（圓內有藍底時）或 `#0ea5e9`（圓為透明／白底時）
- 深色版：圓底 `#0f172a`，折線與節點 `#38bdf8`

**用在 Header 的樣子**
- 水平排列：\[圓形+折線圖標]  MLPredict
- 與網站 primary 色一致，支援淺色／深色 Header。

**縮成 32×32 Favicon 要保留的元素**
- 保留「圓 + 內裡折線與 3 個節點」；圓可略縮以留白邊，避免貼滿 32×32 時顯得擠。
- 捨棄文字。

---

## 3. Favicon 實作（採用概念 C：圓形 + 折線）

- **選用概念**：概念 C（圓形容器 + 極簡折線），辨識度高、在小尺寸下仍清晰。
- **放置目錄**：`/public/favicon.svg`
- **layout / metadata**：在 `app/layout.tsx` 的 `metadata` 中加入 `icons: { icon: '/favicon.svg' }`（見專案內 `src/app/layout.tsx`）。

### SVG 原始碼（favicon）

**放置目錄**：`public/favicon.svg`

```svg
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 32 32" fill="none">
  <circle cx="16" cy="16" r="15" fill="#0ea5e9"/>
  <polyline
    points="8,22 16,12 24,6"
    stroke="#ffffff"
    stroke-width="2"
    stroke-linecap="round"
    stroke-linejoin="round"
    fill="none"
  />
  <circle cx="8" cy="22" r="2" fill="#ffffff"/>
  <circle cx="16" cy="12" r="2" fill="#ffffff"/>
  <circle cx="24" cy="6" r="2" fill="#ffffff"/>
</svg>
```

- **圓形背景**：`#0ea5e9`（與網站 primary 一致）
- **折線 + 節點**：白色，3 點 2 段（左下→中上→右上），縮成 32×32 時仍可辨識。
